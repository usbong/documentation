## Problem & Answer
<b>Problem:</b> Printer prints blank page when using LibreOffice Writer (ver: 6.4.3.2) on LUBUNTU (ver: 20.04)<br/>
<b>Answer:</b> 
1) Go to: Tools, Options, Print<br/>
2) Unset: in Defaults, "PDF as standard print job format"<br/>
<br/>
Done!<br/>
